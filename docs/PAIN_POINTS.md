# Whisperall - "Dolores Reales" (10)

Date: 2026-02-03

This is the short list of user-visible problems that make the app feel non-monetizable today.

1) Provider/model selection dominates the UI
   - Symptom: users feel forced to choose providers/models/params before they can "just dictate".
   - Impact: cognitive overload; feels like a dev tool, not a product.

2) Dropdowns/overlays break layout and readability
   - Symptom: selecting a provider makes the interface look broken (overlay dropdown pushes/overlaps content).
   - Impact: immediate loss of trust.

3) Controls are cramped; key lists are unreadable
   - Symptom: voices lists, provider lists, and settings feel too narrow/truncated.
   - Impact: "I can't even read what I'm selecting".

4) Inconsistent layouts across modules
   - Symptom: each module looks like a different app; spacing, headers, panels, and CTAs move around.
   - Impact: no learnability; users re-learn the UI every time.

5) Repeated UI code = drift
   - Symptom: same concepts implemented 3+ different ways (engine selector, cards, dropdowns, action bars).
   - Impact: bugs and inconsistency grow exponentially with each new feature.

6) Lack of a clear "core product" in navigation
   - Symptom: sidebar lists many modules with similar weight; no hierarchy (Core vs Tools vs Labs).
   - Impact: user doesn't understand what Whisperall is "for".

7) Missing product-grade onboarding
   - Symptom: no guided first-run to achieve "first successful dictation" quickly.
   - Impact: drop-off before users feel value.

8) Missing guided states (model missing, API key missing, GPU unavailable)
   - Symptom: errors feel technical; users don't see the next step.
   - Impact: support burden + churn.

9) Branding/tray polish is weak
   - Symptom: no strong logo/identity; tray icon feels unfinished.
   - Impact: lowers perceived quality and willingness to pay.

10) Monetization narrative is unclear
   - Symptom: many modules exist, but none is presented as the premium "killer workflow".
   - Impact: hard to price; hard to convert; feels like a bundle of experiments.
